# Název projektu

Cinemacity - Ordering system

## Popis
Tento projekt je webová aplikce pro objednávání jídla do kuchyně přes webovou aplikaci. Funguje na principu kde jeden člověk s první stránkou vidí všechny tlačítka s jídlem a když na nějaké klikne tak se na druhé stránce(V kuchyni) objeví obrázek danéhojídla. Tato aplikace vznikla za ůčelem zrychlení výdej jídla a pro větší přehled v objednávkách

## Použití
Aplika běží aktuálně na soukormé webowé stránce pro společnost CinemaCity. V repozitáři je ukázka zdrojového kódu pro ukázku(liší se od použivaného kódu).

## Funkce
- Vkládání objednávek přes `admin.html`.
- Přehled všech objednávek v `public.html`.
- Zvukové efekty při přidání objednávky
- Ukazování času jak dlouho je objednávka objednaná
- Následné odstranění objednávki po její dokončení
